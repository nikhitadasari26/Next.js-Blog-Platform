# Mastering Tailwind CSS

Tailwind CSS significantly speeds up development.

## Utility Classes

- `flex`: Sets display to flex
- `pt-4`: Adds padding top
- `text-center`: Aligns text to center

> Tailwind is awesome!

```css
.btn {
  @apply px-4 py-2 bg-blue-500 text-white rounded;
}
```
